# Cafetería Keeny
Ecommerce y sistema de gestión de recursos.

## Caracteristicas Generales
- Gestión de Usuarios
- Gestión de Bebidas
- Gestión de Comidas
- Gestión de Clientes
- Reporte de Ventas
- Catalogo paginado de bebidas y comidas.
- Carrito de compras

## Tecnologías Utilizadas
- Angular
- Angular Materials
- JSPdf
